# 🤖 ربات تلگرام - راهنمای نصب

## اجرای محلی (لوکال)

```bash
pip install -r requirements.txt
python bot.py
```

---

## دیپلوی رایگان روی Railway

1. برو به [railway.app](https://railway.app) و با گیتهاب لاگین کن
2. **New Project** → **Deploy from GitHub repo**
3. فایل‌های پروژه را آپلود کن
4. در تب **Variables** این متغیرها را اضافه کن:
   - `BOT_TOKEN` = توکن ربات
   - `OWNER_ID` = آیدی عددی اونر
5. روی **Deploy** کلیک کن

---

## دیپلوی رایگان روی Render

1. برو به [render.com](https://render.com) و ثبت‌نام کن
2. **New** → **Background Worker**
3. فایل‌ها را آپلود یا از گیتهاب کانکت کن
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python bot.py`
6. متغیرهای محیطی را اضافه کن

---

## قابلیت‌های ادمین

| قابلیت | توضیح |
|--------|-------|
| 📊 وضعیت | تعداد کاربران، فایل‌ها، ادمین‌ها |
| 📢 پیام همگانی | ارسال هر نوع پیام به همه کاربران |
| 📁 آپلود فایل | آپلود و دریافت لینک اشتراک‌گذاری |
| 🔍 جستجوی فایل | جستجو در فایل‌های آپلودی |
| 🗑 حذف فایل | حذف فایل با توکن |
| 💾 بک‌آپ | ارسال دیتابیس به پیوی اونر |
| 🚫 بن/آن‌بن | مسدود کردن کاربران |
| 👮 مدیریت ادمین | اضافه/حذف ادمین |
| ⚙️ عضویت اجباری | فعال/غیرفعال کردن با یک کلیک |

## اضافه کردن ادمین

در چت با ربات بنویس:
```
addadmin 123456789 username
```

## حذف ادمین
```
removeadmin 123456789
```
