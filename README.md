# 🤖 آپلودر تریاک (Teriak Uploader)

ربات تلگرامی حرفه‌ای و کاملاً فارسی برای آپلود فایل و تولید لینک اختصاصی.
ساخته‌شده با **Aiogram 3** و **SQLite**.

## ✨ قابلیت‌ها

- آپلود فایل توسط ادمین و تولید لینک اختصاصی (`https://t.me/BOT?start=CODE`)
- ذخیره امن فایل‌ها در **Storage Channel** (کانال ذخیره)
- عضویت اجباری چندکاناله با بررسی خودکار و نمایش فقط کانال‌های عضو نشده
- پنل مدیریت کامل با Inline Keyboard و FSM
  - 📊 بررسی وضعیت
  - 📢 پیام همگانی (با شمارش موفق/ناموفق)
  - 📂 مدیریت فایل‌ها (لیست، لینک، اطلاعات، حذف)
  - 📤 آپلود فایل
  - 🔍 جستجوی فایل (نام یا کد)
  - 👥 مدیریت ادمین‌ها
  - 🚫 مدیریت کاربران (بن / آن‌بن / اطلاعات)
  - 📢 مدیریت عضویت اجباری و کانال‌ها
  - 💾 دریافت بکاپ دیتابیس
  - ⚙️ تنظیمات
- منوی کاربر: 📥 دریافت فایل، 📞 تماس با پشتیبانی

## 📋 پیش‌نیازها

- Python 3.12+
- یک ربات تلگرام (توکن از [@BotFather](https://t.me/BotFather))
- یک کانال ذخیره که ربات در آن **ادمین** باشد
